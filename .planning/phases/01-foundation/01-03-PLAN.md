<task type="auto">
  <name>Task 4: Implement Canvas Drawing Logic</name>
  <files>Views/MouseClickView.xaml.cs</files>
  <action>
Create MouseClickView.xaml.cs with drawing logic:

**Fields:**
- Point _dragStartPoint
- bool _isDragging
- Rectangle? _dragPreviewRect

**Methods:**

*OnCanvasMouseLeftButtonDown:*
- If not in adding mode: handle region selection
- If in adding mode: start drag
  - _dragStartPoint = e.GetPosition(canvas)
  - _isDragging = true
  - CaptureMouse()
  - Create preview rectangle (dashed border, semi-transparent)

*OnCanvasMouseMove:*
- If dragging: update preview rectangle size
  - Current point = e.GetPosition(canvas)
  - Width = Current.X - Start.X
  - Height = Current.Y - Start.Y
  - Handle negative values (dragging up/left)

*OnCanvasMouseLeftButtonUp:*
- If dragging: finish region creation
  - Calculate final rect (normalize negative dimensions)
  - Convert to percentages (divide by canvas actual size)
  - Call ViewModel.CreateRegion(x, y, width, height)
  - Remove preview rectangle
  - ReleaseMouseCapture()
  - _isDragging = false

*OnRegionDeleteClick:*
- Click delete button on region
- Call ViewModel.DeleteRegion(region)
- Stop propagation (don't select region)

*Hit Testing:*
- When clicking canvas (not in add mode):
  - Check if click is inside any region (using ClickRegionModel.ContainsPoint)
  - Select top-most region if overlapping
  - Deselect all if clicking empty space

**Aspect Ratio Maintenance:**
- Handle SizeChanged on canvas container
- Recalculate canvas size to maintain aspect ratio
- Center canvas in available space
  </action>
