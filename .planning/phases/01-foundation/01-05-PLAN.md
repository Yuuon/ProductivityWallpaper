   - Each item:

     - Style name below image
     - 12h/24h toggle buttons (horizontal)
     - Opacity slider (0-100%)
     - Checkmark icon in top-right when IsActive

3. **Item Visual States:**
   - Normal: standard border
   - Active: accent color border + checkmark
   - Hover: slight elevation/shadow

**XAML Structure:**
```xml
<ItemsControl ItemsSource="{Binding ClockStyles}">
    <ItemsControl.ItemsPanel>
        <ItemsPanelTemplate>
            <UniformGrid Columns="3" />
        </ItemsPanelTemplate>
    </ItemsControl.ItemsPanel>
    <ItemsControl.ItemTemplate>
        <DataTemplate DataType="{x:Type models:ClockStyleModel}">
            <Border ...
                  BorderBrush="{Binding IsActive, Converter={StaticResource BooleanToActiveBrushConverter}}">
                <Grid>
                    <!-- Checkmark (visible when active) -->
                    <Path Data="..." Visibility="{Binding IsActive, Converter={StaticResource BooleanToVisibilityConverter}}" />
                    
                    <!-- Preview Image -->
                    <Image Source="{Binding PreviewImagePath}" />
                    
                    <!-- Controls -->
                    <StackPanel>
                        <TextBlock Text="{Binding Name}" />
                        <StackPanel Orientation="Horizontal">
                            <Button Content="12h" IsEnabled="{Binding Format, Converter={StaticResource FormatToBoolConverter}, ConverterParameter=Hour12}" />
                            <Button Content="24h" IsEnabled="{Binding Format, Converter={StaticResource FormatToBoolConverter}, ConverterParameter=Hour24}" />
                        </StackPanel>
                        
                        <Slider Value="{Binding Opacity}" Minimum="0" Maximum="1" />
                    </StackPanel>
                </Grid>
            </Border>
        </DataTemplate>
    </ItemsControl.ItemTemplate>
</ItemsControl>
```

**Click Behavior:**
- Click anywhere on item: ToggleClockActivation
- Toggle buttons: SetClockFormat
- Slider: SetClockOpacity (throttled)
