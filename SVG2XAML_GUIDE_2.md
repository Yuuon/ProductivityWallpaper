# SVG图片转化XAML UI 说明文件

阅读AGENTS.md和AGENTS_UI_Workflow.md，根据其对UI处理流程的规范，按需处理以下提及的UI的修改或新建。
所有生成的UI默认语言使用英文词条Key，并补充对应的英文和中文本地化翻译

本文档的主要目标是完成“创作主题”页面的所有UI框架，目前已有首页（主题预览页，CreatorView），现在需要按左侧导航按钮依次添加后续功能页；

* 左侧导航按钮更新
桌面背景，鼠标点击，关机，开机重启，唤醒这五个按钮添加支持多套方案的二级菜单，即：点击某个具有二级菜单的按钮时，展开该按钮功能对应的二级菜单，二级菜单内容为该功能下的所有已创建方案+一个永远在最下方的新建方案按钮（式样与二级菜单内容不统一以作区别），如果当前功能没有任何方案则自动新建一个默认方案。一级菜单的功能按钮右侧添加上或者下的箭头帽以示当前是展开还是收起的状态。多套方案同一时间只能激活一个，当前激活的方案高亮显示并在文字后加一个勾号。左侧导航按钮整体区域追加垂直滚动条，仅当内容超出当前显示区域时显示。

* 多方案右侧页面共通部分 -- 可编辑标题
桌面背景，鼠标点击，关机，开机重启，唤醒这五个按钮对应的右侧页面最上访统一为当前方案名称的标题栏，其中左侧开始为方案名称+编辑按钮（手写图标），点击将该Label转换成TextInput来进行编辑，点击空白处结束编辑并保存，注意保存后同步更新到左侧二级菜单，可编辑标题的具体细节参考主题预览页（CreatorView）的标题编辑功能；标题栏右侧为按钮“激活该方案”，已激活的方案显示“当前激活方案”，如果有变更更新左侧二级菜单激活状态。新建方案时，默认新建名为“桌面背景 x”，x从1开始，重名则累加。

* 桌面背景 （无内容，首次导入）
参考文件："ReferenceResources/创作主题 - 上传文件.svg"。
页面内容分为自上而下四块：最上方标题为共通可编辑标题和可激活按钮（svg没有，新追加内容），然后是上方标题“导入媒体库”，中间区块显示支持的文件和一些限制（支持的文件格式：JPG，PNG，WebP, BMP, HEIC, SVG, GIF, APNG, MP4, MOV, WebM，最大文件大小：500MB，最大分辨率8k(7600x4320)）, 最下面区块显示一个上传云的图标（Resources/Img/Icon-upload.svg），点击导入按钮，以及提示词。

* 桌面背景 （有内容）
参考文件：“ReferenceResources/创作主题 - 编辑文件.svg”。
页面内容分为自上而下四块：最上方标题为共通可编辑标题和可激活按钮（svg没有，新追加内容），然后是上方标题“编辑文件”，同时右侧有三个圆角按钮：继续导入，预览，删除；中间区域为图片/视频导入资源区域，区域内部上方为小标题“图片/视频”，同时右侧有下拉菜单关于播放模式（顺序循环，随机等），下方主体区域为图片/视频显示列表，每一个单元从左到右依次为：序号(以#数字形式从上而下递增)，预览缩略图(图片右下角显示按钮Resources/Img/Icon-preview.svg,视频右下角显示按钮Resources/Img/PlayButton.svg用于预览)，文件格式，文件大小，视频的播放时长（图片留空），声音/静音按钮用于表示视频内容是否静音（图片留空），壁纸在桌面上的显示方式的下拉菜单（填充（推荐），居中，平铺（只限图片）等），移除按钮。声音按钮，下拉菜单，移除按钮三个元素右对齐；下方区域为音频导入资源区域，布局与图片/视频区域类似，区域内部上访为小标题左侧“语音”，右侧是播放模式的下拉菜单，下方主体区域为音频显示列表，每一个单元从左到右依次为：序号(以#数字形式从上而下递增，独立计数)，预览缩略图占位符（中间显示Resources/Img/Icon-music.svg，右下角显示一个用于预览的播放按钮Resources/Img/PlayButton.svg），文件格式，文件大小，音频的播放时长，声音/静音按钮用于表示音频内容是否静音，移除按钮。静音和移除按钮右对齐。静音按钮图标（Resources/Img/Icon-mute.svg）,非静音按钮图标（Resources/Img/Icon-unmute.svg）。

* 桌面背景附加内容 - 预览窗口
参考文件：“ReferenceResources/创作主题 - 编辑文件 - 查看图片.svg”，“ReferenceResources/创作主题 - 编辑文件 - 播放视频.svg”，“创作主题 - 编辑文件 - 播放语音.svg”。
页面窗口比较简单，标题栏上方右侧最小化，最大化，关闭按钮，下方主体区域用于预览，图片显示图片，音频/视频需要内容一个简易播放器，播放音频时中央区域显示音符图标占位符（Resources/Img/Icon-music.svg）。

* 鼠标点击页面
参考文件：“ReferenceResources/创作主题 - 鼠标点击 - 空.svg” （无内容），“ReferenceResources/创作主题 - 鼠标点击 - 点击区域.svg” （无内容，但是有手动创建的鼠标点击区域），“ReferenceResources/创作主题 - 鼠标点击 - 点击区域/桌面背景.svg” （有内容，同时有手动创建的鼠标点击区域），"ReferenceResources/创作主题 - 鼠标点击 - 点击区域/桌面背景/交互内容.svg"（当点击创建好的鼠标点击区域后显示的具体内容）。
页面内容分为自上而下四块：最上方标题为共通可编辑标题和可激活按钮（svg没有，新追加内容），然后是上方标题：“鼠标点击”；中间为鼠标点击区域编辑工作区，首先分为左右两块，左侧为又分为上下两块，上方为标题栏，左端显示“设置点击区域”以及“当前屏幕分辨率[具体分辨率]([屏幕比例])”，右端按钮为点击后进入添加模式的添加按钮（加号图标），下方为编辑区,该编辑区大小与当前屏幕比例保持一致，背景在有可预览的视频/图片时会显示预览内容（没有显示深灰色背景），在该编辑区内，添加模式下点击鼠标并拖动可以绘制矩形，绘制完成的矩形右上角有圆形减号按钮，用于移除该矩形，绘制完一个矩形后退出编辑模式并自动聚焦在该矩形上（高亮显示），需要绘制新的矩形就再点击添加模式按钮添加。非添加模式下，点击已有的矩形则高亮并聚焦在该矩形上。中间区域右侧为单列纵向列表，显示已经导入的图片/视频（即桌面背景窗口中导入的内容），点击列表中的各项目会以“桌面背景中设置的显示方式（即填充，居中，平铺这些）”显示在中间区域左侧的编辑区，即这个编辑区是模拟整体桌面，预览内容以设置好的形式显示在这个模拟的桌面上。整体页面的下方区域为配置交互区，标题“配置交互”下方两个无底色按钮："+导入图片或视频（仅1个）"和"+导入语音（最多5个）"，当点击导入图片或视频时，调用文件管理器选择一个图片/视频导入，使用与桌面背景图片/视频显示具体条目相同的方式显示该项目，并隐藏导入按钮。如果需要替换，需要移除已有项目，还原导入按钮，重新导入；音频同理，不过音频最多支持5个，只有当已有5个时再隐藏导入按钮，当数量小于5个时重新显示。

这三个页面布局与桌面背景页面相同，只有在实际功能上有区别（即只有激活时点不同）

* 桌面时钟
参考文件：“ReferenceResources/创作主题 - 桌面时钟.svg”。
附加小功能，显示当前所有的桌面时钟式样，点击某个式样激活该桌面时钟并在右上角显示圆形勾号图标，点击已激活的桌面时钟取消激活。每个项目里面有12小时 24小时两个按钮选项，以及一个透明度拖动条。

* 番茄钟
参考文件：“ReferenceResources/创作主题 - 番茄专注钟.svg”。
附加小功能，与桌面时钟类似，只不过换成带有计时功能的番茄钟，同时有toggle选项：开启免打扰模式，下方为tooltips。

* 倒数纪念日
参考文件：“ReferenceResources/创作主题 - 倒数纪念日 - 空.svg”和“ReferenceResources/创作主题 - 倒数纪念日 - 有内容.svg”。
主体区域分为上中下三块：上方为小标题“倒数纪念日”，中间为“桌面显示样式”，这里显示所有的倒数纪念日样式，和时钟/番茄钟一样，点击某个样式以激活，再点击取消激活；下方以grid列表的形式显示所有已经设定的纪念日，每个纪念日可以自定义名称（名称的标题右侧放删除按钮），然后设置重复模式，再就是选择日期。不重复显示完整日期，每年重复只选择月/日，每月只选择日，每周则选择星期几。最下方永远是按钮“+新增纪念日”，点击则新增一个上述纪念日项目。
